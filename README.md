# weintern-internship

Code Structure & Design Decisions:

My webpage uses semantic HTML5 (<header>, <nav>, <main>, <section>, <footer>) for better accessibility and SEO. The CSS implements mobile-first responsive design with Flexbox/Grid layouts and a @media (max-width: 768px) breakpoint that stacks all columns vertically on mobile devices. I chose Google Fonts (Inter + Poppins) for clean typography and used WeIntern's brand colors (#0A1628 Navy, #2563EB Blue, #F59E0B Gold) throughout. The JavaScript adds interactive elements like a mobile menu toggle, back-to-top button, and CTA button alerts for better user experience. The four required sections (Hero, About, Internship, CTA) are clearly separated, with value proposition cards and internship tracks displayed in responsive grids that reorganize on smaller screens.
